Customer Engagement

Monitoring Services

Service summary

To support the service delivery processes, Cegeka monitors various aspects of the different components (availability, capacity, security, database backups) via the central monitoring platform. All monitoring information is accessible to our technicians and to certain employees of the Customer and is visualized via a central uniform web interface. Service windows and maintenance windows are defined per managed system (this information is centrally stored and updated in the Cegeka CMDB). A link between the Cegeka CMDB and the monitoring platform ensures that the monitoring platform is always working with the latest information. This also ensures that the service levels are very clear to both the support teams and the service delivery manager.

Standard service components

Cegeka is responsible for providing the following services:

  • Monitoring of standard OS features (e.g. availability, performance, security and configuration);
  • Monitoring of technical applications (e.g., Active Directory);
  • Monitoring of application services, as defined by the Customer;
  • URL monitoring, as defined by the Customer;
  • Setting up and accessing Cegeka's central customer monitoring portal.

Security

The Service is certified against ISO27001. Processes for managing this service are part of the yearly ISAE3000 statement.
