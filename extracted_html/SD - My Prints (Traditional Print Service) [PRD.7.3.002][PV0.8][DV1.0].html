Print service traditional (My Prints)

Service summary

The My Prints service delivers the management and operation for the printer queues of all supported and contracted network printers, multi-functionals and plotters. Through this Service we manage the local printing infrastructure from a technical perspective to support printing functionality for authorized users.

This Service includes the operational monitoring and administration for the print management service infrastructure.

Standard service components

Cegeka shall be responsible in providing the following Services: 

  • Print driver management: With this Service, Cegeka will deliver and install print drivers, for all different workplace versions and operating systems in scope;
  • Print queue management: Cegeka will manage the different print queues, operating systems (of the central print management service infrastructure), software and other assets in order to provide seamless printing experience.

Optional service components

Integration with third-party print solutions which enable:

  • Follow-me functionality;
  • Secure printing solutions;
  • Scanning.

Eligibility and Prerequisites

Conditions

  • OS Management service by Cegeka is mandatory for this service;
  • The availability of a system to function as Print Server;
  • A functional Active Directory.

Out of scope

  • The provisioning of printers, multifunctional devices and associated supplies are not included in this Service.
